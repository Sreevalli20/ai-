# 🚀 ET Smart Life Concierge  
AI-Powered Financial & Life Decision Assistant  

---

## 📌 Problem Statement  
Most individuals struggle with managing their finances and life decisions due to:
- Lack of financial awareness  
- No personalized guidance  
- Multiple fragmented tools  
- Time-consuming decision-making  

95% of users do not have a structured financial plan.

---

## 💡 Solution  
**ET Smart Life Concierge** is a multi-agent AI system that:
- Understands user profile (income, expenses, goals)  
- Calculates financial health score  
- Provides personalized financial advice  
- Recommends insights and actions  

It acts as a **personal AI advisor** for smarter decisions.

---

## 🧠 System Architecture  

The system uses a **multi-agent AI architecture**:

- 👤 **User Profiling Agent** → Collects user data  
- 📊 **Money Health Agent** → Calculates financial score  
- 💰 **Finance Agent** → Generates financial advice  
- 📰 **News Agent** → Provides personalized insights  
- 🚀 **Action Agent** → Suggests next steps
